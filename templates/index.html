Who SMS App

This simple application looks up the Wikipedia page for a particular person and returns the first few sentences. Incoming messages must be in the format who [person name].

Examples

who michael jackson
who sting
who kagame

Sawanet Configuration

In order for this application to work with Sawanet, you'll need to configure your keyword to post to the URL for this application. For this app that URL is: http://who.aws.af.cm/sms/receive?text=%(text)s&number=%(number)s You'll also need to set the delivery method to be POST and to check the Delivery Response option.
